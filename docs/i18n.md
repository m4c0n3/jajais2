# Internationalization (i18n)

## Defaults
- Default locale: `sk`
- Fallback locale: `en`

## User locale
- Users can store a preferred locale in `users.locale`.
- If set, requests use that locale; otherwise the app uses the default.

## Admin language switch
- In the admin panel user menu, pick "Slovensky" or "English".
- This updates the user preference and session locale.

## Module translations
- Modules can ship translations in:
  - `modules/<Module>/lang/<locale>/*.php`
- Access translations with namespace:
  - `HelloWorld::messages.greeting`

## Testing fallback
- When a key is missing in `sk`, Laravel falls back to `en`.
